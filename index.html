FastForward. HR менеджмент тур

Верстал как тестовое задание и для практики. Landing page для регистрации на HR-менеджмент тур. Разметка по БЭМ, адаптивная резиновая страница, для сеток использованы Bootstrap и flex'ы. Проект собирается и оптимизируется с помощью Gulp, использовал LESS для CSS, непрописаные ссылки отключает скрипт, настроена проверка и отправка формы на почту.
